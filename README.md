# easy-rabta

## Frontend env

`easy-rabta/.env.local`:

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:5000
```

## Run frontend

```bash
npm install
npm run dev
```

Login page: `http://localhost:3000/login`

## Backend setup (separate folder)

Backend folder: `../backend`

1. `backend/.env` banayein from `.env.example`
2. Backend run karein:

```bash
cd ../backend
npm install
npm run dev
```
